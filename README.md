# Домашняя работа по блоку 16.2

## Описание:

Домашняя работа по блоку 16.2. Множественное наследование. Реализованы абстрактные классы для класса Product и его 
наследников, Category и Order, так же добавлены абстрактные методы, которые переопределяются в Product, Category, Order.
Добавлен класс-миксин, который при создании объекта класса, печатает в консоль информацию о том, от какого класса и с 
какими параметрами был создан объект. Реализован новый класс Order, в котором будет ссылка на то, какой товар был 
куплен, количество купленного товара, а также итоговая стоимость.
Написаны тесты для новых функциональностей, уровень покрытия 92%.
